***********************帮助*********************** 
考虑的燃耗链: U234->U235->U236 U238->U239->Np239->Pu239->Pu240->Pu241->Pu242
考虑的裂变产物: I135->Xe135
外步长：指整个燃耗时间段的划分，每点击一次‘计算’，程序自动计算一个外步长的时间，并停下来等待用户修改通量、功率等参数。
内步长：指燃耗方程数值求解过程中的步长，影响程序计算的精度。
定通量燃耗：指在所划分的燃耗时间段内中子通量水平保持不变。
定功率燃耗：指在所划分的燃耗时间段内反应堆功率水平保持不变，程序自动在每个内步长上根据当前核密度的变化调整中子通量。
初值：只有第一步计算时可输入各核素核密度的初值，之后每步计算都自动以上一步计算的结果作为初值。
核数据：来源于ENDF，能量0.0253eV，用户可根据需求在工具栏‘修改核数据’中进行修改。
